npm init --yes    -> Inicializar un proyecto de Node
npm install express --save    -> Instalar un paquete, el(--save) es para guardarlo en las dependencias
node index.js   -> Arrancar el proyecto node

npm i -D nodemon    -> Instala nodemon como una dependencia de desarrollo de manera local
En el archivo package.json, en el grupo de escripts, agregar lo siguiente:
"dev": "nodemon index.js"
Para ejecutarlo, escribir en la terminal:
npm run dev

npm i mongoose --save    -> Para instalar el componente mongoose que provee los métodos para acceder a la base de datos MongoDB

npm i dotenv --save     -> Para crear variables de entorno personalizadas se instal este componente para no exponer los datos de conexión.


se debe crear el archivo llamado ".env" obligatoriamente en la carpeta madre

copiar la Uri de conexion de la DB y pegarlo despues del MONGODB_URI = del .env (no dejar espacio despues del =), luego copiar la contraseña autogenerada del usuario admin en Mongo en la parte de la Uri que dice <db_password> y quitar tambien los "<>"

alt + shift + f para formatear el codigo y quitar caracteres raros de copiar y pegar codigo de otro lados
